
You can get Doxygen at:
http://www.doxygen.org

You can get DOT for doxygen at:
http://www.graphviz.org

You should create an environment variable named DOXYGEN and point it
to the bin sub-dir of your doxygen installation directory

Once you have taken these steps you can use MSWIN_MakeDocs.bat to
generate the doxygen documentation for you
