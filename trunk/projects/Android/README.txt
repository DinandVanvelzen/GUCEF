


=== Ant
To install Ant on Ubuntu Linux just run "SudoGetAnt.sh"
On windows download ant and place it in the archive dependencies folder
Check this website for downloading Ant: http://ant.apache.org/bindownload.cgi

=== NDK & SDK
Make sure you have the Android NDK and SDK in the archive dependencies folder
Always try and use the latest NDK and update your SDK
NDK/SDK website: http://developer.android.com/sdk/ndk/index.html
